Atenção!

É importante você ter algumas coisas já instaladas para continuarmos:

Instale o gort: https://gort.io/documentation/getting_started/downloads/

Instale o gem: sudo apt install gem

Atualize os pacotes: sudo apt update

Atualize os diretórios: sudo apt upgrade

Instação do ruby

Siga esses simples passo a passo e você terá o ruby na sua máquina:

Instale o ruby na sua máquina: sudo apt install ruby-full

Caso queira o pacote mais simples do ruby, execute: sudo apt install ruby essential

Caso o pacote acima não venha instalado o gem, execute: sudo apt install gem

Logo após instalar o gem, execute: sudo gem install rubygems

Pronto, o ruby já está na sua máquina!

Instalação da IDE Arduino

Siga esses simples passos e você terá a IDE Arduino na sua máquina:

A IDE Arduino é fácil de ser instalada e pode ser obtida através do link: https://www.arduino.cc

Entretanto, você pode usar o seguinte comando no terminal para instalar também: sudo apt install arduino

Pronto, a IDE Arduino já está na sua máquina!

Instalação do framework Arduino_Firmata

Siga esses simples passos e você terá o framework na sua máquina:

Instale o framework na sua máquina: sudo gem install arduino_firmata

Atenção: tente conectar o arduino com o ruby, caso não funcione, execute e tente novamente: sudo gem install arduino

Você pode consultar e saber mais no link: https://rubygems.org/gems/arduino_firmata

Sobre versões: https://rubygems.org/gems/arduino_firmata/versions

Pronto, o framework já está na sua máquina!

Instalação o seu ambiente de desenvolvimento

Siga esses simples passos e você terá o ambiente de desenvolvimento na sua máquina:

Nós aconselhamos a você usar o ambiente de desenvolvimento feito pela Microsoft Corporation, o Visual Studio Code

Você pode obter ele através do link: https://code.visualstudio.com

Entrentanto, você pode instalar o VS Code via terminal usando os seguintes comandos:

Instale o snap: sudo apt install snap

Instale o VS Code através do snap: sudo snap install code

Pronto, o ambiente de desenvolvimento já está na sua máquina!
